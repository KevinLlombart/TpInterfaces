El objetivo general de este curso es capacitar a los estudiantes en el desarrollo de aplicaciones utilizando el framework .NET de Microsoft. Al finalizar el curso, los estudiantes serán capaces de:

  • Comprender los fundamentos de la programación orientada a objetos.
  • Desarrollar aplicaciones de escritorio utilizando Windows Forms y WPF.
  • Crear aplicaciones web dinámicas con ASP.NET y MVC.
  • Utilizar el IDE de Visual Studio de manera eficiente.
